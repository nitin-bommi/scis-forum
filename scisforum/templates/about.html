Welcome to SCIS Forum

Introduction

We wanted to provide you the unsatisfied requirements by moodle:

  • Quick session time-outs 😤
  • Unattractive CSS 🤒
  • No dark mode - Hmm... light attracts bugs and we are allergic to bugs 😨
  • No communication between different course students 😢
  • Time table clashes 😫
So we came up with a solution -

"...A unique shared platform for our school."

What we included

  • Shared time-table for all users 📆
    • Read access to students
    • Read-write access to faculty
      • (You need to email us or write in the suggestion box to test this feature...)
  • Forum ✍️
  • NLP to filter-out unparliamentary words 🤖
  • Login with face 🧑
  • Dark mode feature - no more bugs 🐞
  • Clean UI 🎨

We hope that we surprised you!
